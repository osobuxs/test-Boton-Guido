POC WhatsApp Embedded Signup — Índice

Página simple para navegar a las pantallas del POC.
Recomendación: servilo por http(s) (Vercel / Live Server), no como file://.

1) Pantalla principal (POC)

Botón “Conectar WhatsApp” → abre popup → recibe token via postMessage → permite guardar integración.
Abrir wa-poc.html

2) Bridge (popup retorno)

La abre el backend al final del callback. Consulta /meta/oauth/poll, hace postMessage al opener y cierra.
(No la abras “a mano” salvo para debug)
Abrir wa-bridge.html (debug)

3) Links backend (debug rápido)

Endpoints útiles para probar el flujo desde el navegador.
Abrir /backoffice/whatsapp/signup/config Abrir /meta/oauth/poll?state=...
Tip: primero pedí config para obtener un state, después probás poll.

4) Checklist

Si algo falla:
  • Popups habilitados
  • META_REDIRECT_URI == callback real del backend
  • El dialog usa redirect_uri igual al callback
  • El exchange usa el mismo redirect_uri
  • App Domains / Valid OAuth Redirect URIs bien

Si querés, después lo dejo más minimal todavía (un solo botón y un logcito).
